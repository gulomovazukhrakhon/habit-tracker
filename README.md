# Overview

The application allows users to:

- Create custom habits
- Manage and edit habits
- Mark habits as completed
- Review predefined habits
- Generate reports and analytics
- Delete habits manually or automatically

---

# Features

## Habit Management
- Create new habits
- Edit habit names
- Change habit frequency
- Mark habits as completed
- Delete habits manually

## Analytics & Reports
- View all habits
- View daily habits
- View weekly habits
- View a specific habit
- Analyze tracked habits

## Additional Features
- Interactive CLI menu
- Predefined habits
- Pretty formatted tables
- Automatic deletion after expiration
- Modular OOP architecture

# 📦 Requirements

Create a `requirements.txt` file with:

```txt
prettytable==3.5.0
questionary~=1.10.0
pytest~=7.2.0 
```
---
# Installation

1.  Clone the Repository
```bash 
git clone https://github.com/gulomovazukhrakhon/habit-tracker.git
```
2. Navigate to the Project Folder
```bash 
cd habit-tracker
```
3. Install Dependencies
```bash 
pip install -r requirements.txt
```
4. Running the Application
```bash 
python main.py
```
